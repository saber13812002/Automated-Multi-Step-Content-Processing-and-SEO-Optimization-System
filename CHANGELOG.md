## [0.1.0] - 2024-12-19

### Initial Release

- Automated subtitle generation system
- WordPress category crawling and processing
- Media download automation
- Language segmentation pipeline
- ChromaDB search API with Redis caching
- Multi-step content processing workflows
- n8n integration support
- Support for multiple languages (Farsi, Arabic, etc.)
